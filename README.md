# software-development <br>
This repo consists of notes of software development subject of DIT- 5th semester<br>
Syllabus for this subject<br>
<br>
Software Development<br>
EG3102IT<br>
<br>
Course objectives:<br>
After completion of this course students will be able to:<br>
      1. Explain the theory and foundations of software development.<br>
      2. Apply the concept of different phase of system development life cycle while
      designing a software.<br>
      3. Draw the Context diagram, DFD and ER Diagram of a system.<br>
      4. Apply fact-finding and problem-solving skills.<br>
